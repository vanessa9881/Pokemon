Project Title:

Detective Pikachu Card Set Web App. Able to Create backup
of Detective Pikachu Pokemon card sets.

Prerequisites:

1.Download files from github

2.have node installed on machine

3.Have mysql installed on machine


Deployment:

1.Open terminal or command line

2.write command: "node server.js"

3.Open a browser and proceed to: http://localhost:8080


Built With:

Language: javascript(node.js)

Framework: Express

Database Management System: mysql
